[X] grid: 16 x 12
[X] kolom 1, 5, 9 & 13 ander kleurtje
[X] cellen aanklikbaar
[X] run! (highlight row)
[ ] controls: stop/start Icons with http://fontawesome.io/
[ ] controls: bpm instellen
[ ]
[ ] muziek (instrument)

[ ] meerdere patterns
[ ] drum of melodie pattern
[ ] song arranger
[ ] instrument (voice) editor
